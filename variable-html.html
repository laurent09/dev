Liste complète des attributs et éléments HTML

  • id : Identifie un élément de manière unique dans une page.
  • class : Attribue une ou plusieurs classes à un élément pour le styliser ou le manipuler avec CSS ou JavaScript.
  • style : Permet d'ajouter des styles CSS directement à un élément.
  • src : Spécifie la source d'un élément externe, comme une image, une vidéo ou un script.
  • href : Définit l'URL vers laquelle un lien pointe.
  • alt : Fournit un texte alternatif pour les images (utile pour l'accessibilité et si l'image ne se charge pas).
  • title : Ajoute une info-bulle qui s'affiche lorsque l'utilisateur survole l'élément.
  • value : Définit la valeur d'un élément, comme un champ de formulaire.
  • placeholder : Affiche un texte indicatif dans un champ de formulaire.
  • disabled : Désactive un élément (souvent utilisé dans les formulaires).
  • required : Rend un champ de formulaire obligatoire.
  • type : Définit le type d'un élément, comme un champ de formulaire.
  • target : Spécifie où ouvrir un lien (par exemple, dans un nouvel onglet).
  • data-* : Permet de stocker des données personnalisées sur un élément.
  • aria-* : Attributs pour améliorer l'accessibilité (ARIA = Accessible Rich Internet Applications).
  • lang : Définit la langue du contenu d'un élément.
  • tabindex : Définit l'ordre de tabulation des éléments.
  • contenteditable : Rend un élément éditable par l'utilisateur.
  • hidden : Masque un élément.
  • autofocus : Focalise automatiquement un élément lorsque la page est chargée.
  • div : Conteneur générique pour regrouper des éléments.
  • span : Conteneur en ligne pour regrouper du texte ou des éléments.
  • p : Définit un paragraphe.
  • a : Crée un lien hypertexte.
  • img : Affiche une image.
  • ul : Crée une liste non ordonnée.
  • li : Élément d'une liste (utilisé avec ul ou ol).
  • ol : Crée une liste ordonnée.
  • table : Crée un tableau.
  • tr : Définit une ligne dans un tableau.
  • td : Définit une cellule de données dans un tableau.
  • th : Définit une cellule d'en-tête dans un tableau.
  • form : Crée un formulaire.
  • input : Crée un champ de saisie.
  • button : Crée un bouton.
  • textarea : Crée une zone de texte multiligne.
  • select : Crée une liste déroulante.
  • option : Définit une option dans une liste déroulante.
  • header : Définit l'en-tête d'une section ou d'une page.
  • footer : Définit le pied de page d'une section ou d'une page.
  • section : Définit une section dans un document.
  • article : Définit un contenu indépendant, comme un article de blog.
  • nav : Définit une section de navigation.
  • main : Définit le contenu principal d'un document.
  • aside : Définit un contenu secondaire ou une barre latérale.
